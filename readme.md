# Azure Static Web App Demo

Мінімальний проєкт для деплою через GitHub Actions у Azure Static Web Apps.

## Структура

- `index.html` — головна сторінка
- `style.css` — стилі
- `script.js` — логіка
- `api/data.json` — тестове API (статичний файл)

## Завдання студентів

1. Змінити HTML (додати секцію / картки).
2. Змінити CSS (кольори, шрифти, layout).
3. Змінити JSON і перевірити новий деплой.
4. Додати ще один "API файл" у папку `/api`.
5. Змінити JS: показувати більше даних.
6. Створити тег і подивитися, як запускається деплой.
7. Перевірити лог деплою в Azure.
